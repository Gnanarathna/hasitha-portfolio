# Portfolio Website

This is my personal portfolio website, showcasing my skills, projects, and experience as a software developer. It's built with modern web technologies and designed to be a clean and effective presentation of my work.

## Live Demo

You can view the live version of my portfolio here: [https://hasitha-anjana-portfolio.vercel.app](https://hasitha-anjana-portfolio.vercel.app)

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A fast build tool and development server for modern web projects.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
- **Framer Motion**: A library for creating production-ready animations.
- **React Icons**: A collection of popular icon libraries for React.
- **EmailJS**: A service for sending emails directly from JavaScript.

## Features

- **Responsive Design**: The website is fully responsive and works on all devices.
- **Smooth Animations**: Interactive elements and page transitions are animated with Framer Motion.
- **Dynamic Typing Effect**: An engaging typing animation in the hero section.
- **Project Showcase**: A dedicated section to display my projects with links to their demos and source code.
- **Contact Form**: A functional contact form that sends emails directly to me.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js and npm installed on your machine.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Gnanarathna/hasitha-portfolio.git
   ```
2. Navigate to the project directory
   ```sh
   cd hasitha-portfolio
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
